3. PCA Based Bands Selection¶

Preprocessed images will be used for selecting the most important bands out of all bands in given images.

Step to be followed are ..

  • Preparation of eigen vectors and eigen values and save it into directory in .csv format
  • Reload eigen statistics to decide maximum number of PCs to be selected for bands' selection
  • Selection of three different sets of bands included in top N contributing bands with min_freq=2 where N = 10, 25, 50

Band Reduction usig PCA results¶

  • pca_result_directory : path to the file which contains all the PCA output
  • top : Number bands to be considered as of top contributing bands.
  • max_pc : Maximum number of PCs to be considered for pca method. For e.g. max_pc = 2 means take mean of firat two pcs
  • min_frequency : Minimum number of ocuurences of a band in top N contributing bands of all the images.
